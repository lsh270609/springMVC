URL
	사용자가 서버에 접속해서 서비스를 받기 위해 입력하는 주소
	
	형식
	protocol://domainaddres:portnumber/경로1/경로2/...
		ㄴ 프로토콜 : server 와 client 사이의 통신을 위한 규약
		   domain address(도메인주소) : IP 주소를 사람이 알아보기 쉬운 이름을 사용해서 표현함
		   port number(포트 번호) : 1 ~ 65535 번 사이의 숫자로 구성. 컴퓨터 내에서 포트번호로 프로그램을 구분함. 생략 시 80으로 설정됨.
		   경로1 == project 이름 == contextPath
